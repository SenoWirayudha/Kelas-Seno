BELAJAR HTML DAN CSS

HTML digunakan untuk membuat tampilan website, CSS digunakan untuk mengatur tampilan

MATERI HTML

  • h1 untuk judul
  • p untuk paragraf
  • img untuk gambar
  • table untuk tabel
  • input untuk form
  • li untuk list atau data

MATERI CSS

  1. margin, padding, position untuk mengatur jarak antara elemen HTML
  2. display untuk mengatur elemen HTML agar tampil atau tidak tampil, mendatar (kolom), atau menurun (baris)
  3. style untuk mengatur font
